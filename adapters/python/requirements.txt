# The hinzu Python adapter's resolution backend.
#
# ty (Astral's Rust type checker), driven over its LSP, is the adapter's SOLE
# resolution backend. It is a separate binary, not a Python package dependency —
# install it however you like: `uv tool install ty`, or `pip install ty` (this
# file). When the `ty` binary is on PATH the adapter uses it automatically
# (`HINZU_TY` overrides the path). If `ty` is absent the adapter exits nonzero
# with an honest message — there is no fallback resolver.
#
# `ast` is in the standard library, so the adapter has no Python-package
# dependency of its own; this pin makes the backend reproducible in CI.
ty==0.0.61
